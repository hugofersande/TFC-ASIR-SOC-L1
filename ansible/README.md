# Automatización con Ansible

Esta carpeta queda preparada para una futura automatización del despliegue.

## Objetivo
Permitir una instalación reproducible de:

- Wazuh Server
- Elastic Stack
- TheHive
- Cortex
- MISP
- Agentes en endpoints

## Futuro contenido recomendado

```text
ansible/
├── inventories/
├── group_vars/
├── roles/
│   ├── wazuh/
│   ├── elastic/
│   ├── thehive/
│   ├── cortex/
│   ├── misp/
│   └── agents/
└── site.yml
```
